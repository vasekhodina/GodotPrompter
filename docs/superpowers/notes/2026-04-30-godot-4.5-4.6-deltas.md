# Godot 4.5/4.6 Deltas — Per-Skill Inventory

Working document for the v1.5.0 alignment pass (Task 7). Each row pairs a Godot 4.5 or 4.6 feature with the skill(s) that should mention it, plus a source URL (release notes article or merged PR). Rows without a stable, verified source URL were dropped.

## Research sources

- Godot 4.5 release page: https://godotengine.org/releases/4.5/
- Godot 4.5 beta 1 announcement: https://godotengine.org/article/dev-snapshot-godot-4-5-beta-1/
- Godot 4.6 beta 1 announcement: https://godotengine.org/article/dev-snapshot-godot-4-6-beta-1/

Godot 4.6 had reached **beta 1** as of this writing. It is included here with `(Godot 4.6+)` annotations; those entries should be marked accordingly in skills so readers know they require 4.6.

v1.4.0 already covered: localization (new skill), procedural-generation (new), xr-development (new), gdscript-patterns (super(), typed dicts), audio-system (interactive music streams, runtime WAV loading), ai-navigation (async nav baking), animation-system (LookAtModifier3D, SpringBoneSimulator3D, retargeting), shader-basics (Compositor effects), state-machine (HSM/parallel FSM), 2d-essentials (TileMapLayer deprecation note), physics-system (Jolt as default). Those are **not** repeated here.

| gdscript-patterns | 4.5 | Variadic functions — GDScript functions now accept an arbitrary number of trailing arguments using the `...args` syntax, collected as an Array. | shader-basics | 4.5 | Stencil buffer support — All rendering backends (Vulkan, D3D12, Metal, GLES3) now expose stencil write/read operations via render modes and shader built-ins (`stencil_write_mode`, `stencil_read_mode`, `stencil_value`, etc.) on spatial and canvas_item shaders. Enables portals, X-ray vision, outline masks, and holes-in-geometry effects that were impossible before. | https://github.com/godotengine/godot/pull/80710 | Stencil Buffer Effects |
| shader-basics | 4.5 | SMAA 1x antialiasing — Sub-pixel Morphological Antialiasing is now a built-in post-processing AA mode. Enable via **Project Settings → Rendering → Anti Aliasing → Quality → Screen Space AA → SMAA**. Provides sharper results than FXAA. Can be combined with MSAA or TAA. | https://godotengine.org/releases/4.5/ | SMAA Antialiasing |

The existing 4.4 note in the skill (which says "default for new 3D projects in 4.4") should be updated to reflect 4.6 solidifying this as stable/non-experimental. | https://godotengine.org/article/dev-snapshot-godot-4-6-beta-1/ | (update to existing section 8 — Jolt Physics note) |
